INTRODUCTION
------------

The Identity Proof Field Types module has a single custom field focussing on ID
proof. This Field can be attached to most content where there is a need of ID
proof type field.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/idproof

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/idproof


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Install the Identity Proof Field Types module as you would normally install a
   contributed Drupal module. Visit https://www.drupal.org/node/1897420 for
   further information.


CONFIGURATION
-------------

    1. Navigate to Administration > Extend and enable the module.
    2. Navigate to Administration > Structure > Content types > [Content type to
       edit] > Manage fields and add a new field.
    3. Select ID Proof from the dropdown. Save and continue.
    4. Select the ID type for the default with any needed description. Save
       settings.
